# App Inventaire QR Code

## Description

Application web de gestion d’inventaire permettant le suivi des biens d’une organisation via un système de QR Code.  
Elle permet la gestion des utilisateurs, des services, des biens, ainsi que la génération de documents (PDF) et le suivi des mouvements (entrées et sorties).

---

## Architecture du projet

- Frontend : React  
- Backend : Node.js + Express  
- Base de données : MySQL  

---

## Installation et lancement

Pour lancer le projet, il faut exécuter les commandes suivantes :

cd backend  
npm install  
npm start  

cd frontend  
npm install  
npm start  

---

## Fonctionnalités

- Gestion des utilisateurs (authentification)
- Gestion des rôles (admin, chef, utilisateur)
- Ajout, modification et suppression des biens
- Gestion des services et départements
- Système d’inventaire complet
- Génération de QR Code pour les biens
- Génération de PDF (inventaires, procès-verbaux, états)
- Gestion des entrées et sorties de biens
- Upload de fichiers (images, documents)
- Tableau de bord administrateur et chef de service
- Historique et traçabilité des actions

---

## Objectif

Projet développé dans le cadre d’un stage, ayant pour objectif de digitaliser la gestion d’inventaire et d’améliorer le suivi des biens au sein d’une organisation via un système de QR Code.
